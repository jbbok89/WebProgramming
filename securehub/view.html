[Injection] SQL 인젝션 분석 및 방어
CVE-2024 | 홍길동 | 2026-04-09
SQL 인젝션(SQL Injection)은 악의적인 SQL 구문을 입력 필드에 삽입하여 데이터베이스를 조작하는 공격입니다. 공격자는 로그인 폼이나 검색창에 특수 SQL 구문을 입력하여 인증 우회, 데이터 탈취, 심지어 데이터 삭제까지 수행할 수 있습니다. 방어 방법: - Prepared Statement / Parameterized Query 사용 - 입력값 유효성 검증 및 이스케이프 처리 - 최소 권한 원칙 적용 - WAF(Web Application Firewall) 도입
