■ PIE (Position-Independent Executable)

코드 영역까지 무작위 주소에 배치하여 실행 파일의 위치를 예측할 수 없게 만드는 보안 기술입니다.


[ ASLR과의 비교 ]

- ASLR: 스택, 힙, 라이브러리 주소만 무작위화

- PIE: 실행 파일의 코드 영역까지 포함하여 무작위화


[ 작동 원리 ]

PIC(Position-Independent Code)를 기반으로 하며, 상대 참조 방식을 사용하여 어느 주소에서든 코드가 정상 실행되도록 지원합니다.
